- VBRP, 대금 청구 문서 : 품목 데이터,
- VBAG, 판매 문서 : 일정 라인 별 데이터 릴리스,

- VBKD, 영업 문서 : 비즈니스 데이터,

- VBKPF, 문서 주차 용 문서 헤더,
